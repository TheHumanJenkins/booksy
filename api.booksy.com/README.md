## Backend

To get started:

1. run `npm install`
2. create a `.env` file
3. Populate the following environment variable:
   - <b>PORT</b>=3001
   - <b>GUSTO_CLIENT_ID</b>=${YOUR_GUSTO_CID}
   - <b>GUSTO_CLIENT_SECRET</b>=${YOUR_GUSTO_CSECRET}
4. run `npm start`
